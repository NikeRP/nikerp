УСТАНОВКА
================================================
В архиве присутствует две версии автомобиля. Версия с раскрасками (добавляются под именем coquette_sign_#.png в архив с текстурами)
или версия с игровыми цветами. Выбирайте на ваш вкус.

Файлы coquette.yft, coquette_hi.yft, coquette.ytd и vehicles_coquette_interior.ytd заменить по следующему адресу:
D:\Program Files\Grand Theft Auto V\x64e.rpf\levels\gta5\vehicles.rpf\

Файл coquette_mods.rpf заменить по следующему адресу:
D:\Program Files\Grand Theft Auto V\x64i.rpf\levels\gta5\vehiclemods\

Файл vehicles.meta заменить по следующим адресам:
1. D:\Program Files\Grand Theft Auto V\common.rpf\data\levels\gta5
2. D:\Program Files\Grand Theft Auto V\update\update.rpf\common\data\levels\gta5

Если вы до этого момента изменяли значения в данных файлах, то необходимо открыть эти файлы через блокнот и при помощи поиска
со значением coquette найти строку со значениями FLAGS и заменить ее на строку указанную ниже:

<flags>FLAG_SPORTS FLAG_SPAWN_ON_TRAILER FLAG_EXTRAS_STRONG FLAG_PARKING_SENSORS FLAG_RICH_CAR FLAG_NO_BROKEN_DOWN_SCENARIO FLAG_RECESSED_TAILLIGHT_CORONAS FLAG_HAS_LIVERY</flags>

Эту операцию необходимо совершить с обоими файлами vehicles.meta!!!
